üß™ Test These Containers Locally

Option 1: Interactive Python Console

# Run interactive Python in container docker run -it --rm python:3.11-alpine python # Inside Python container, try: # >>> import sys # >>> sys.version # >>> print("Hello from Docker Python!") # >>> exit()

Option 2: Redis Database

# Start Redis container docker run -d -p 6379:6379 --name test-redis redis:alpine # Connect to Redis (in another terminal) docker exec -it test-redis redis-cli # Inside Redis, try: # > SET greeting "Hello from Docker Redis!" # > GET greeting # > exit # Clean up docker stop test-redis && docker rm test-redis

Option 3: Node.js REPL

# Run interactive Node.js docker run -it --rm node:alpine # Inside Node container, try: # > console.log("Hello from Docker Node!"); # > process.version # > .exit

Option 4: Alpine Linux Shell

# Get shell inside Alpine Linux container docker run -it --rm alpine:latest /bin/sh # Inside container, try: # $ cat /etc/os-release # $ whoami # $ ls -la # $ exit

üîç Verify Real Containers

These commands prove you're running real Docker containers:
