üß™ Test These Containers Locally

Option 1: Nginx Web Server (Port 8080)

# Pull and run nginx container docker run --rm -d -p 8080:80 --name test-nginx nginx:alpine # Check it's running docker ps # Visit http://localhost:8080 in your browser # Stop when done docker stop test-nginx

Option 2: Hello World Container

# Run hello-world container (outputs text and exits) docker run --rm hello-world # This will download and run the official hello-world image

Option 3: Interactive Container

# Run interactive Alpine Linux container docker run --rm -it alpine:latest /bin/sh # You'll get a shell inside the container # Type 'exit' to leave

üîç Verify Real Containers

These commands prove you're running real Docker containers:

# See all running containers docker ps # See Docker images on your system docker images # Check container resource usage docker stats # View container logs docker logs test-nginx

üåç Production Deployment

On the server, these containers run automatically via the deploy script:

# Server deployment command (in deploy.sh) sudo docker run -d \ --name egle-nginx-demo \ -p 8080:80 \ --restart unless-stopped \ nginx:alpine # Check if it's running on server sudo docker ps curl localhost:8080
